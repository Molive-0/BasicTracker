Detailed Description

Gui handling code and rudimentary API.

It helps the MainWindow create the GUI for the program. It might not be totally needed but it helps organise the code. I think. It was originally meant to handle a lot of the GUI code for multiple windows, like clipping and spawning and such. As there is only one window this is redundant to an extent.
