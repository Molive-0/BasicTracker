\doxysubsection{Detailed Description}
Gui handling code and rudimentary A\+PI. 

It helps the \mbox{\hyperlink{class_basic_tracker_1_1_main_window}{Main\+Window}} create the G\+UI for the program. It might not be totally needed but it helps organise the code. I think. It was originally meant to handle a lot of the G\+UI code for multiple windows, like clipping and spawning and such. As there is only one window this is redundant to an extent.
